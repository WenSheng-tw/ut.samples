OAuth para UT
=============
Acceso a OAuth2 desde UT. Este ejemplo esta basado en el ejemplo de Diego para mod harbour v2.1
https://github.com/mod-harbour/mod_harbour.v2/blob/master/Samples/goauth/readme.txt

IMPORTANTE: Para poderlo ejecutar necesitaras configurar tu sitio google tal como se explica 
y crear tu fichero de configuracion credentials.json.

Para este ejemplo la url de redireccion que habras de configurar en google es 
http:/localhost/oauth_access

Accessing OAuth2 from UT. This example is based on Diego's example for mod harbour v2.1
https://github.com/mod-harbour/mod_harbour.v2/blob/master/Samples/goauth/readme.txt
